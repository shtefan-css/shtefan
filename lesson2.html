Урок 1: Что такое CSS?

CSS (Cascading Style Sheets) — это язык стилей, используемый для описания внешнего вида HTML-документов.

CSS позволяет разделить структуру и внешний вид документа, делая веб-страницы более гибкими и управляемыми.

Пример применения CSS

p {
  color: blue;
  font-size: 16px;
}
